Detailed Description

JSON-file-backed repository for Appointment entities.

Persists appointments to src/main/resources/storage/appointments.json instead of a relational database, keeping scheduling data separate from the H2/PostgreSQL schema. Auto-assigns sequential integer IDs on first save.
