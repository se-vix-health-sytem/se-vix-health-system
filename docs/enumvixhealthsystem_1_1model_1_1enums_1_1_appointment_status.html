Detailed Description

Lifecycle state of an vixhealthsystem.model.medical.Appointment.

Transitions flow roughly as: PENDING → CONFIRMED → COMPLETED, with CANCELLED or RESCHEDULED as possible side exits. Business rules for allowed transitions are enforced in vixhealthsystem.model.medical.Appointment.
