Detailed Description

Core entity services — employees, patients, and departments.

  • EmployeeService — full employee lifecycle: create, update, delete, plus Firebase account provisioning and password resets
  • PatientService — patient registration, lookup, and profile updates
  • DepartmentService — department queries used by public site pages (specialist listings, department detail views)
