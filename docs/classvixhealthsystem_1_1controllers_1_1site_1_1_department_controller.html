Detailed Description

Controller for the public department directory — base URL /departments.

Exposes a browseable listing of all clinical departments and a detail page for each one, showing the assigned doctors, the services offered, and a profile image map. The Administration department is hidden from the public list because it has no patient-facing services.

String departmentDetail ( @PathVariable String id,
Model model )

GET /departments/{id} — display the detail page for a single department.

Loads the department by its string identifier, then separately queries the doctors, services, and doctor image map that belong to it. Using the department's numeric Long ID for the service calls avoids exposing the surrogate key in the URL path.

Parameters
idString identifier of the department (as used in URLs).
modelReceives department, doctors, services, doctorImages, and pageTitle attributes.
Returns
Thymeleaf template departments/detail, or redirect:/departments when the identifier is not found.
