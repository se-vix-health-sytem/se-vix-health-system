Detailed Description

Append-only JSON-file-backed repository for AuditLog entries.

Persists audit logs to src/main/resources/storage/audit-logs.json. Intentionally exposes no delete or update methods to enforce NFR02 (Traceability) — once written, a log entry is immutable for normal users. The save(AuditLog) method is synchronized to prevent ID collisions under concurrent request handling.
