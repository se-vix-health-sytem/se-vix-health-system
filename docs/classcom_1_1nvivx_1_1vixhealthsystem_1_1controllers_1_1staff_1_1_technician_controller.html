Detailed Description

Controller for Technician staff members — base URL /technician.

Technicians maintain the hospital's machinery fleet. This controller covers viewing all machines, filtering by faulty or maintenance status, updating machine status, logging repairs, viewing active alerts, and the technician's own shift and profile pages.

Only accessible to users with ROLE_TECHNICIAN.

Use cases covered: UC26 (machinery maintenance), UC27 (status updates), UC28 (alerts), UC30 (profile/shifts).
