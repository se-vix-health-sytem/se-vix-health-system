Detailed Description

Spring Security authority roles assigned to every authenticated user in the system.

The ROLE_ prefix is required by Spring Security's default role-naming convention and maps directly to the hasRole() expressions used in com.nvivx.vixhealthsystem.config.SecurityConfig.
