Detailed Description

Data access layer — Spring Data JPA repositories and JSON-backed stores.

Most repositories here are standard Spring Data interfaces that extend JpaRepository. The JSON-backed ones (prefixed Json…) manage entities that are stored in flat files rather than the SQL database — this was a design choice for data that changes frequently and doesn't need relational integrity (appointments, shifts, vacation requests, audit logs).

JPA repositories: DepartmentRepository, EmployeeRepository, MachineryRepository, MedicalConditionRepository, MedicalFacilityRepository, MedicalRecordRepository, PatientRepository, PrescriptionRepository, ResourceRepository, RoomRepository, StorageRepository, SurgeryRepository

JSON-backed stores: JsonAppointmentRepository, JsonAuditLogRepository, JsonShiftRepository, JsonVacationRepository
