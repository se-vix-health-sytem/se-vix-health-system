Detailed Description

Simulates outbound patient notifications : appointment confirmations, reminders, exam result alerts, and cancellations.

No real email is sent; each method logs a [DEMO EMAIL] line so the behaviour is visible in the console during development. Replacing the log calls with an actual mail sender (SMTP / SendGrid / etc.) is all that's needed to make these live.
