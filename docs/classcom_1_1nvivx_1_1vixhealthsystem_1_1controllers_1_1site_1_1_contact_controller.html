Detailed Description

Controller for the public contact form — no base URL prefix.

Presents a simple contact form and handles its submission. In the current implementation the message is logged to the application log rather than forwarded to an email service; this is a deliberate placeholder that keeps the demo self-contained without requiring SMTP configuration.
