Detailed Description

Physical structure of the hospital system.

Models the real-world hierarchy: a MedicalFacility (hospital or clinic) contains Departments, which own Rooms, an Office, and optionally a Storage unit. Rooms can be specialised (SpecializedRoom, InternationRoom).

Location holds the address used on public-facing pages (maps, contact info). Hospital extends MedicalFacility and is the concrete type persisted in the database.
