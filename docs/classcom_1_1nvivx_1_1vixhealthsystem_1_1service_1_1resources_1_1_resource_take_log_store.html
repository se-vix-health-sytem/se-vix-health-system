Detailed Description

JSON-backed append log that records every resource takeout event performed by employees.

Entries are stored in src/main/resources/storage/resource-take-logs.json and are sorted newest-first on retrieval. All public methods are synchronized to prevent concurrent write corruption from multi-threaded request handling.

Used by InventoryService#removeResourceFromStorage(com.nvivx.vixhealthsystem.model.person.employee.Employee, Long, int) to maintain a separate, human-readable consumption audit trail alongside the main com.nvivx.vixhealthsystem.service.AuditService log.
