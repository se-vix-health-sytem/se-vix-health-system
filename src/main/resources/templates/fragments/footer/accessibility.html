Accessibility Statement

VIX Health System is designed to be accessible and user-friendly for all users, including people with disabilities.

User Friendliness (NFR08)

The interface is simple, intuitive, and designed for quick access to core functions like booking and viewing records.

Keyboard Navigation

All main features are accessible via keyboard navigation.

Responsive Design

The system works on mobile, tablet, and desktop devices.

Readable Design

High contrast colors and readable fonts are used to improve accessibility.

Continuous Improvement

We continuously improve usability to meet accessibility standards (WCAG principles).
