Accessibility Statement

Commitment to Accessibility
VIX Health System is committed to ensuring digital accessibility for all users, including people with disabilities.
User Friendliness (NFR08)

The interface is simple, intuitive, and designed for quick access to core functions like booking and viewing records. We follow WCAG 2.1 Level AA guidelines to ensure an inclusive experience for all users.

Keyboard Navigation

All main features are accessible via keyboard navigation. You can navigate through the site using:

  • Tab — Move between interactive elements
  • Shift + Tab — Move backwards
  • Enter — Activate buttons and links
  • Space — Check/uncheck checkboxes
  • Esc — Close modals and menus
Responsive Design

The system works seamlessly on mobile, tablet, and desktop devices. The layout adapts to different screen sizes and orientations, ensuring all content remains accessible regardless of device.

Readable Design

High contrast colors and readable fonts are used to improve accessibility:

  • Minimum contrast ratio of 4.5:1 for normal text
  • Font sizes are adjustable using browser zoom (up to 200% without loss of content or functionality)
  • Clear, readable typography using system fonts
  • Sufficient spacing between interactive elements
Screen Reader Compatibility

The system is designed to work with major screen readers including:

  • NVDA (Windows)
  • JAWS (Windows)
  • VoiceOver (macOS, iOS)
  • TalkBack (Android)

We use semantic HTML, proper ARIA labels, and logical heading structures to ensure content is properly interpreted by assistive technologies.

Alternative Text

All meaningful images include descriptive alternative text. Decorative images are marked as such and ignored by screen readers.

Form Accessibility

All forms include:

  • Properly associated labels for each input field
  • Clear error messages with instructions for correction
  • Logical tab order
  • Required field indicators
Time Limits

Users are automatically logged out after 8 minutes of inactivity for security. An on-screen warning is displayed 1 minute before logout, giving users the option to extend their session.

Focus Indicators

Clear visual focus indicators are provided for all interactive elements, making it easy to identify which element currently has keyboard focus.

Language & Readability

The system uses clear, plain language whenever possible. Technical medical terms are explained when necessary. The primary language is set to Italian, with English available for international staff.

Accessibility Testing

We continuously improve usability to meet accessibility standards (WCAG principles). Our testing includes:

  • Automated accessibility testing with industry-standard tools
  • Manual keyboard navigation testing
  • Screen reader testing
  • User testing with people who have disabilities
